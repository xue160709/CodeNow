# 图片生成

文生图、图生图（参考主体）。`POST https://api.minimaxi.com/v1/image_generation`

鉴权：`Authorization: Bearer <MINIMAX_API_KEY>`

## 请求体（常用字段）

| 字段 | 类型 | 说明 |
| :--- | :--- | :--- |
| `model` | string | 如 `image-01` |
| `prompt` | string | 画面描述 |
| `aspect_ratio` | string | 如 `16:9` |
| `response_format` | string | 如 `base64` |
| `subject_reference` | array | 图生图：参考图，元素含 `type`（如 `character`）、`image_file`（URL） |

成功时从 `response.json()["data"]["image_base64"]` 取 Base64 列表。

## 文生图

```python
import base64
import os
import requests

url = "https://api.minimaxi.com/v1/image_generation"
headers = {"Authorization": f"Bearer {os.environ['MINIMAX_API_KEY']}"}

payload = {
    "model": "image-01",
    "prompt": "A cat on a windowsill, soft daylight, photorealistic",
    "aspect_ratio": "16:9",
    "response_format": "base64",
}

r = requests.post(url, headers=headers, json=payload)
r.raise_for_status()
images = r.json()["data"]["image_base64"]

for i, b64 in enumerate(images):
    with open(f"output-{i}.jpeg", "wb") as f:
        f.write(base64.b64decode(b64))
```

## 图生图（主体参考）

```python
import base64
import os
import requests

url = "https://api.minimaxi.com/v1/image_generation"
headers = {"Authorization": f"Bearer {os.environ['MINIMAX_API_KEY']}"}

payload = {
    "model": "image-01",
    "prompt": "女孩在图书馆窗前看向远方",
    "aspect_ratio": "16:9",
    "subject_reference": [
        {
            "type": "character",
            "image_file": "https://example.com/reference.jpg",
        }
    ],
    "response_format": "base64",
}

r = requests.post(url, headers=headers, json=payload)
r.raise_for_status()
images = r.json()["data"]["image_base64"]

for i, b64 in enumerate(images):
    with open(f"output-{i}.jpeg", "wb") as f:
        f.write(base64.b64decode(b64))
```
