# 视频生成

异步流程：**创建任务** → **轮询状态** → **按 file_id 取下载地址并下载**。

| 步骤 | 方法 | URL |
| :--- | :--- | :--- |
| 创建任务 | `POST` | `https://api.minimaxi.com/v1/video_generation` |
| 查询状态 | `GET` | `https://api.minimaxi.com/v1/query/video_generation?task_id=...` |
| 获取文件信息 | `GET` | `https://api.minimaxi.com/v1/files/retrieve?file_id=...` |

鉴权：`Authorization: Bearer <MINIMAX_API_KEY>`

## 模式与主要参数

1. **文生视频**：`prompt`、`model`、`duration`、`resolution` 等  
2. **图生视频**：在上述基础上增加 `first_frame_image`（URL）  
3. **首尾帧**：`first_frame_image`、`last_frame_image`、`prompt` 等（模型需支持）  
4. **主体参考**：`subject_reference`（如 `type: character` + `image: [url]`），常用模型如 `S2V-01`  

部分模型支持在 `prompt` 中使用镜头类指令（如 `[运镜]`），详见平台正式文档。

## 完整示例（Python）

```python
import os
import time
import requests

api_key = os.environ["MINIMAX_API_KEY"]
headers = {"Authorization": f"Bearer {api_key}"}


def invoke_text_to_video() -> str:
    url = "https://api.minimaxi.com/v1/video_generation"
    payload = {
        "prompt": "镜头拍摄一位女性坐在咖啡馆里，抬头望向窗外，镜头缓慢移向窗外街道，暖色调，氛围轻松。",
        "model": "MiniMax-Hailuo-2.3",
        "duration": 6,
        "resolution": "1080P",
    }
    r = requests.post(url, headers=headers, json=payload)
    r.raise_for_status()
    return r.json()["task_id"]


def invoke_image_to_video() -> str:
    url = "https://api.minimaxi.com/v1/video_generation"
    payload = {
        "prompt": "Contemporary dance, the people in the picture are performing contemporary dance.",
        "first_frame_image": "https://example.com/first-frame.png",
        "model": "MiniMax-Hailuo-2.3",
        "duration": 6,
        "resolution": "1080P",
    }
    r = requests.post(url, headers=headers, json=payload)
    r.raise_for_status()
    return r.json()["task_id"]


def invoke_start_end_to_video() -> str:
    url = "https://api.minimaxi.com/v1/video_generation"
    payload = {
        "prompt": "A little girl grow up.",
        "first_frame_image": "https://example.com/start.jpeg",
        "last_frame_image": "https://example.com/end.jpeg",
        "model": "MiniMax-Hailuo-02",
        "duration": 6,
        "resolution": "1080P",
    }
    r = requests.post(url, headers=headers, json=payload)
    r.raise_for_status()
    return r.json()["task_id"]


def invoke_subject_reference() -> str:
    url = "https://api.minimaxi.com/v1/video_generation"
    payload = {
        "prompt": "人物在古老石板路上行走，自然光，电影感镜头运动。",
        "subject_reference": [
            {
                "type": "character",
                "image": ["https://example.com/face-reference.png"],
            }
        ],
        "model": "S2V-01",
        "duration": 6,
        "resolution": "1080P",
    }
    r = requests.post(url, headers=headers, json=payload)
    r.raise_for_status()
    return r.json()["task_id"]


def query_task_status(task_id: str) -> str:
    url = "https://api.minimaxi.com/v1/query/video_generation"
    params = {"task_id": task_id}
    while True:
        time.sleep(10)
        r = requests.get(url, headers=headers, params=params)
        r.raise_for_status()
        body = r.json()
        status = body["status"]
        print(f"status: {status}")
        if status == "Success":
            return body["file_id"]
        if status == "Fail":
            raise RuntimeError(body.get("error_message", "video generation failed"))


def fetch_video(file_id: str, out_path: str = "output.mp4") -> None:
    url = "https://api.minimaxi.com/v1/files/retrieve"
    r = requests.get(url, headers=headers, params={"file_id": file_id})
    r.raise_for_status()
    download_url = r.json()["file"]["download_url"]
    video = requests.get(download_url)
    video.raise_for_status()
    with open(out_path, "wb") as f:
        f.write(video.content)
    print(f"saved: {out_path}")


if __name__ == "__main__":
    task_id = invoke_text_to_video()
    # task_id = invoke_image_to_video()
    # task_id = invoke_start_end_to_video()
    # task_id = invoke_subject_reference()

    file_id = query_task_status(task_id)
    fetch_video(file_id)
```

## 轮询说明

- 建议间隔约 **10 秒**，避免过高请求频率。  
- `status == Success` 时记录 `file_id`；`Fail` 时读取 `error_message`。
