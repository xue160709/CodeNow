# 音乐生成

- **作曲 / 演唱**：`POST https://api.minimaxi.com/v1/music_generation`
- **歌词（可选）**：`POST https://api.minimaxi.com/v1/lyrics_generation`
- **翻唱预处理（两步翻唱步骤 1）**：`POST https://api.minimaxi.com/v1/music_cover_preprocess`

鉴权：`Authorization: Bearer <MINIMAX_API_KEY>`，`Content-Type: application/json`

## 歌词生成（可选）

`mode` 示例：`write_full_song`。有现成歌词可跳过此步。

```python
import os
import requests

url = "https://api.minimaxi.com/v1/lyrics_generation"
headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {os.environ['MINIMAX_API_KEY']}",
}
payload = {"mode": "write_full_song", "prompt": "一首欢乐的新年歌曲"}

r = requests.post(url, json=payload, headers=headers)
print(r.text)
```

## 音乐生成

常用字段：`model`（如 `music-2.6`）、`prompt`（风格）、`lyrics`（歌词，可与上一步衔接）；无词时可查官方文档中 `lyrics_optimizer`、`is_instrumental` 等参数。

```python
import json
import os
import requests

url = "https://api.minimaxi.com/v1/music_generation"
headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {os.environ['MINIMAX_API_KEY']}",
}

payload = {
    "model": "music-2.6",
    "prompt": "Mandopop, Festive, Upbeat, Celebration, New Year",
    "lyrics": "[Verse 1]\n示例歌词第一行\n\n[Chorus]\n示例副歌",
    "audio_setting": {
        "sample_rate": 44100,
        "bitrate": 256000,
        "format": "mp3",
    },
    "output_format": "url",
}

r = requests.post(url, headers=headers, json=payload)
print(json.dumps(r.json(), ensure_ascii=False, indent=2))
```

音频地址在响应 JSON 中（如 `output_format` 为 `url` 时）。

## 翻唱：一步模式

传入参考音频 URL，歌词由服务从音频提取。

```python
import json
import os
import requests

url = "https://api.minimaxi.com/v1/music_generation"
headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {os.environ['MINIMAX_API_KEY']}",
}

payload = {
    "model": "music-cover",
    "audio_url": "https://example.com/original-song.mp3",
    "prompt": "爵士风格，慵懒深夜酒吧，萨克斯",
    "audio_setting": {
        "sample_rate": 44100,
        "bitrate": 256000,
        "format": "mp3",
    },
    "output_format": "url",
}

r = requests.post(url, headers=headers, json=payload)
print(json.dumps(r.json(), ensure_ascii=False, indent=2))
```

## 翻唱：两步模式（可改词）

### 1. 预处理

返回示例字段：`cover_feature_id`（约 24h 有效）、`formatted_lyrics`、`structure_result`、`audio_duration`。此步是否计费以平台说明为准。

```python
import json
import os
import requests

url = "https://api.minimaxi.com/v1/music_cover_preprocess"
headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {os.environ['MINIMAX_API_KEY']}",
}
payload = {"model": "music-cover", "audio_url": "https://example.com/original-song.mp3"}

r = requests.post(url, headers=headers, json=payload)
result = r.json()
cover_feature_id = result["cover_feature_id"]
print(cover_feature_id)
print(result.get("formatted_lyrics", ""))
```

### 2. 使用 `cover_feature_id` 生成

> 使用 `cover_feature_id` 时不要同时传 `audio_url` 或 `audio_base64`。`lyrics` 必填，长度需满足接口限制（如 10–1000 字符，以官方文档为准）。

```python
import json
import os
import requests

url = "https://api.minimaxi.com/v1/music_generation"
headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {os.environ['MINIMAX_API_KEY']}",
}

modified_lyrics = "[Verse 1]\n修改后的歌词\n\n[Chorus]\n修改后的副歌"

payload = {
    "model": "music-cover",
    "cover_feature_id": cover_feature_id,
    "lyrics": modified_lyrics,
    "prompt": "爵士风格，慵懒深夜酒吧，萨克斯",
    "audio_setting": {
        "sample_rate": 44100,
        "bitrate": 256000,
        "format": "mp3",
    },
    "output_format": "url",
}

r = requests.post(url, headers=headers, json=payload)
print(json.dumps(r.json(), ensure_ascii=False, indent=2))
```
